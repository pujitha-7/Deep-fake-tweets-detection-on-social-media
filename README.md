# Deep-fake-tweets-detection-on-social-media
Deepfake detection system using CNN and LSTM models to classify real and fake media content.
## Project Overview

This project focuses on detecting deepfake content using deep learning techniques. 
The system uses CNN for feature extraction and LSTM for sequence learning to classify media as real or fake.

## Features
- Image and video deepfake detection
- CNN-based feature extraction
- LSTM-based temporal analysis
- High accuracy classification

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## Use Case
This project helps in identifying fake media content and preventing misinformation.
